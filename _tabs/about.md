这是我个人的 [GitHub 主页](/)。

## 目的

主要用于记录些平时所学、所思、所感。

* 一方面用于深化理解
* 另一方面作为回顾查询
* 同时也希望能够与他人分享。

## 构建个人 GitHub 主页

如何构建个人的 GitHub 主页，可查看[GitHub Pages 官网](https://pages.github.com/)。本人是采用 [Jekyll](https://jekyllrb.com/) 来构建站点的，这也是 GitHub Pages 推荐的方式。

本人对比了多款Jekyll主题后，采用的是[jekyll-theme-chirpy](https://github.com/cotes2020/jekyll-theme-chirpy)，因为这个主题在满足我个人需要的基础上显得最简洁，而且吃西瓜的小蚂蚁很可爱，哈哈！
